# Bigger Than Business - Walk-In Barber Landing Page

A brutalist-style landing page for Bigger Than Business barbershop in Pflugerville, Texas.

## Design Style

- **Pure brutalist aesthetic**: Black and white with neon green accents
- **Thick borders**, sharp edges, no rounded corners
- **Bold typography** with oversized headlines
- **Grid-based layouts** with heavy visual contrast
- **System fonts** (Helvetica/Arial) for raw, confident feel

## SEO Optimization

- Primary keyword: "walk-in barber Pflugerville"
- Meta title and description optimized for local search
- Semantic HTML structure with proper heading hierarchy
- Location-specific content throughout

## Features

- Fully responsive (mobile-first design)
- Single HTML file with embedded CSS
- Fast loading with no external dependencies
- High-converting copy focused on same-day service
- All CTAs funnel to main booking site

## Usage

Simply open `index.html` in a browser or deploy to any static hosting service (Vercel, Netlify, GitHub Pages).

## Customization

- Accent color: Change `--accent` in CSS (`:root` selector)
- Logo URL: Update `<img>` src attributes
- Booking URL: Replace `https://www.biggerthan-business.com/` links
